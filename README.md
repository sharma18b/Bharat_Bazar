## Run Locally
- Atlas Cloud MongoDB
  - Create database at [https://cloud.mongodb.com](https://cloud.mongodb.com)
  - Create .env file in root folder
  - Set MONGODB_URL=mongodb+srv://your-db-connection

### 3. Run Backend

$ npm install
$ npm start

### 4. Run Frontend

# open new terminal

$ cd frontend
$ npm install
$ npm start

## You can open Port 3000.
